# Day 3 - Expense Tracker with SQLite

## Overview

This project is a command-line expense tracker built with Python and SQLite.

## Features

- Add expenses
- View all expenses
- Filter expenses by category
- Calculate total spending
- Show category-wise summary
- Delete expenses by ID
- Update expenses by ID
- Validate date format
- Export expenses to CSV

## Tech Used

- Python
- SQLite
- CSV module
- Object-oriented design
- SQL queries

## Why this project matters

This project demonstrates practical backend concepts:

- database integration
- CRUD-style operations
- modular project structure
- input validation
- reporting logic
- file export support
